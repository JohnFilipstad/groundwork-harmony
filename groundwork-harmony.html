Typography

Harmonize provides a consistent typographic scale using design tokens. All headings use the --font-family-headings token with appropriate sizes and weights.

Heading Level 1 - The Largest Display

Primary page titles use --font-size-xxl with increased margin for visual hierarchy.

Heading Level 2 - Major Sections

Section headers use --font-size-xl for clear content division.

Heading Level 3 - Subsections

Subsection headers use --font-size-l for nested content organization.

Heading Level 4 - Content Groups

Smaller headings begin to approach body text size but maintain visual weight.

Heading Level 5 - Minor Headings

These use the base font size with different weight for subtle emphasis.

Heading Level 6 - Smallest Headers

The smallest heading uses uppercase transformation for distinction.

Code with Syntax (using code inside pre)

// JavaScript Example
function harmonize(elements) {
  const tokens = getComputedStyle(document.documentElement);

  elements.forEach(element => {
    element.style.setProperty('--primary', tokens.getPropertyValue('--color-primary'));
    element.classList.add('harmonized');
  });

  return elements.length;
}

What is Groundwork Harmony?

Groundwork Harmony is a modern normalization library that provides consistent styling for HTML elements across all browsers. It's built on a token-based design system and focuses on accessibility, performance, and modern CSS features.

Key features include:

  • Modular architecture
  • Design tokens
  • Dark mode support
  • Print optimization
